# 🎙️ ComfyUI F5-TTS Clean Portable Build (RTX 50xx Optimized)

Brought to you by **Node Niño Labs** 🚀

This is an ultra-clean, portable standalone build of ComfyUI pre-configured with **F5-TTS Voice Cloning** nodes. It is specifically optimized to solve CUDA environment conflicts and `libtorchcodec` errors on next-gen NVIDIA **RTX 5050 / 50xx series** GPUs under Windows.

## ✨ Key Features:
* **Zero-Configuration:** Portable structure. Just extract and run.
* **Next-Gen GPU Ready:** Pre-configured Python environment for RTX 5050.
* **FFmpeg Full-Shared Integrated:** Built-in shared DLLs to prevent `torchcodec` audio loading loops.
* **SillyTavern Ready:** Optimized workflows for instant AI character voice generation.

## ☁️ Облачная альтернатива (Для слабых ПК)
Если у вас нет локальной видеокарты серии RTX 50xx, вы можете запустить наш официальный облачный фикс-сервер для SillyTavern в один клик без ошибок совместимости пакетов Python и без конфликтов с `torchcodec`:

👉 **[ЗАПУСТИТЬ NODE NIÑO LABS XTTS FIX В GOOGLE COLAB](https://google.com)**

1. Download **`Node_Nino_Labs_F5TTS_RTX50xx.zip`** from this repository.
2. Extract the archive to `C:\ComfyUI_Clean`.
3. Drop your custom SD/Checkpoint models into `ComfyUI/models/checkpoints/`.
4. Launch the server using **`run.bat`**.
5. Open `http://127.0.0.1:8188` in your browser and start cloning!
